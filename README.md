# 🎯 Cartera Nacho · Tablero 2026

Tablero interactivo (100% estático, sin backend) para planificar una cartera de
inversión desde Argentina operando en **IOL (InvertirOnline)**: CEDEARs, Dólar MEP,
bonos, ON, FCI money market y exposición cripto.

- Sliders que reparten la cartera y muestran los pesos por mes en vivo.
- Presets por **perfil** (conservador / moderado / arriesgado) × **plazo** (corto / mediano / largo).
- Calculadora de **fondo de mudanza** y plan por etapas.
- Referentes en castellano (IG / YouTube) verificados.

> ⚠️ Herramienta **educativa y personal**, no es asesoramiento financiero.
> Toda inversión puede perder valor. Los números son ilustrativos.

Deploy: GitHub Pages (rama `main`, raíz).
